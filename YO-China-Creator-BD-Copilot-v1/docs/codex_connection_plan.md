# Codex + 聚星网站 Skill 连接计划

## 目标链路
GitHub Repo → Codex 读取 CODEX.md + knowledge + workflows → 用户提供达人消息 / Codex 从聚星 skill 获取消息 → Codex 生成回复草稿 → 用户确认 → Codex 调用聚星网站 skill 发送

## 第一阶段
只让 Codex 读取仓库并生成草稿。

## 第二阶段
让 Codex 调用聚星网站 skill 获取待回复消息，但仍只生成草稿。

## 第三阶段
用户明确确认后，Codex 调用发送 skill。

## 发送前检查
- 收件人正确吗？
- 语言正确吗？
- 报价正确吗？
- 合同版本正确吗？
- 是否有不该承诺的内容？
