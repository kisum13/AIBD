# Contract Workflow

## Net Revenue
Required points: deductions + platform dashboard + revenue statement + written request within two months.

## Non-compete
Required points: not exclusive for all AI; project-specific; highly similar AI character/persona/project only.

## Image Rights
Required points: only this project + related promotion/distribution; no unrelated commercial use; no AI model training.

## Contract Term
Default two years. If user says revised contract is one year, follow user instruction.

## MG / Upfront
Revenue Share has no minimum guarantee. Fixed fee has two installments. No payment before production begins unless user provides exception.
