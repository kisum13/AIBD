# 项目介绍

## 核心定义
This is an AI Short Drama project where we co-create an original short drama with the creator as the main character.

它不是传统广告，而是一部围绕创作者本人形象、账号风格和受众定制的 AI 短剧。

## 标准项目结构
- 完整短剧通常 10–20 集。
- 前约三分之一作为免费剧集发布，用于吸引粉丝。
- 后续剧集跳转平台，通过一次性付费解锁整部剧。
- 平均客单价约 USD 3.3 per series。
- 项目制作周期通常 3–6 周。

## 创作流程
1. 创作者提供题材、角色、世界观、氛围方向。
2. 内容团队基于方向制作故事大纲。
3. 创作者审核大纲和剧本，提出反馈。
4. AI 制作团队制作完整短剧。
5. 免费剧集 / 预告 / 精彩片段等发布到创作者社媒。
6. 完整剧集在平台上线并进行全球分发。

## 样片说明
样片仅为参考格式，不是最终项目。每位创作者都会重新定制原创故事。
